Alert status

When enabled, this alert runs on its schedule and is included in “check all enabled.” When off, it stays paused (you can still use “check now” on the dashboard for a one-off run).

When off, matches are still found and shown under “Last run” on the dashboard, but no Pushover/Telegram message is sent and trips are not marked as already notified—so turning this back on later can still alert you for the same trips.
